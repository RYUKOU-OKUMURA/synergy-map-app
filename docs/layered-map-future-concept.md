# レイヤー型シナジーマップ将来構想

作成日: 2026-05-22

## このドキュメントの位置づけ

このドキュメントは、MVP-1後の開発で検討する「レイヤー型シナジーマップ」の構想メモである。

MVP-1の実装範囲を広げるための仕様書ではない。MVP-1では、引き続き「資料投入 → AI抽出カード → 人間確認 → 顧客導線マップ → Markdown / CSV出力」を中心価値として完成させる。

ただし、この構想は将来の継続利用、差別化、横展開に関わる重要な方向性であるため、今後の設計判断で参照できるように残す。

## コアコンセプト

会社全体や事業全体のマップを見るだけで終わらず、気になる導線や詰まりを選ぶと、より細かい粒度へ自然に深掘りできる。

最終的には、抽象的な経営課題を具体的な行動まで降ろし、「じゃあ今はこれをやらなきゃね」と判断できる状態に着地させる。

代表的な体験は次の通り。

```text
全体マップを見る
気になる導線をクリックする
「この導線を深掘り」する
事業単位・業務フロー・タスク単位に分解される
「じゃあ今やるのはこれだね」に着地する
```

この構想は、単なるマップ作成機能ではなく、会社全体から今日やる具体アクションまで降りていける意思決定支援として扱う。

## 想定するマップ粒度

現時点では、次の5つの粒度を想定する。

| 粒度 | 主な対象 | 見える価値 |
| --- | --- | --- |
| 会社・組織マップ | 事業、商品、集客、売上導線 | 全体像、未接続シナジー、戦略上の詰まり |
| 事業単位マップ | 1つの事業、商品群、チャネル、顧客導線 | 伸ばすべき導線、弱いチャネル、改善候補 |
| プロジェクトマップ | 目的、関係者、成果物、期限、リスク | 進行上の詰まり、役割不足、判断点 |
| 業務・ワークフローマップ | 手順、担当、ツール、入力、出力、受け渡し | ムダ、属人化、自動化候補、品質のばらつき |
| タスクマップ | 入力、作業、判断、確認、出力、次アクション | 今日やること、選択肢、確認事項 |

粒度ごとに、抽出するノードの種類と線の意味は変わる。一方で、情報源、確度、情報充実度、詰まり、未接続、施策候補、確認質問といった基本構造は共通化できる可能性が高い。

## レイヤービュー構想

通常表示では、顧客導線ビューのシナジーマップとして表示する。

将来的には、ボタンや表示切り替えによって、同じマップを抽象度別のレイヤーとして見られるようにする。

```text
レイヤー4: 会社全体 / 事業ポートフォリオ
レイヤー3: 事業 / 商品 / 集客導線
レイヤー2: 業務フロー / 顧客対応 / 運用プロセス
レイヤー1: タスク / 担当 / 今日やること
背景: 資料・メモ・データ根拠
```

この表示は、単なる演出ではなく、「今見ている問題がどの抽象度で詰まっているのか」を理解するための視点切り替えとして扱う。

実装方式は未確定とする。初期案としては、本格的な3Dではなく、React Flowなどの2Dデータを保ったまま、擬似2.5Dの積層表示や視点切り替えで表現する方針が現実的である。

## MVP-1との関係

この構想はMVP-1では実装しない。

MVP-1では、以下を優先する。

- 資料やメモを入れやすいこと
- AI抽出結果を人が確認、修正、確定できること
- 顧客導線ビューのマップに確実に到達できること
- 根拠、推測、要確認が区別できること
- 施策候補と確認質問が出ること
- Markdown / CSVで実務に使える出力ができること

ただし、将来この構想へ拡張しやすいように、MVP-1の設計判断では次を意識する。

- マップ対象の粒度を将来増やせる余地を残す
- ノード、エッジ、確度、出典、情報充実度の共通モデルを安易に事業マップ専用に固定しすぎない
- AI抽出テンプレートを将来差し替え、追加できる余地を残す
- マップ表示とドメインロジックを分離し、表示モードを増やせる余地を残す

ただし、これらはMVP-1のスコープを膨らませる理由にはしない。

## 将来の検討課題

Beta以降で検討する課題は次の通り。

- 粒度ごとのノード種別
- 粒度ごとのエッジの意味
- 粒度ごとのAI抽出テンプレート
- 「この導線を深掘り」操作のUX
- レイヤービューの表示方式
- 通常ビュー、深掘りビュー、レイヤービュー、アクションビューの関係
- 深掘り結果の保存単位
- 出力形式にレイヤー構造をどう反映するか
- どのフェーズで導入するか

現時点では、DBカラム、Reactコンポーネント、AI出力JSON schema、詳細な画面遷移、正式なリリース時期までは決めない。
