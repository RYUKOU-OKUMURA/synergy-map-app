# MVP-1 実装計画

作成日: 2026-05-16

## 目的

MVP-1では、最小の業務価値を一気通貫で実装する。

中心価値:

> 資料投入 → AI抽出カード → 人間確認 → 1枚の顧客導線マップ → Markdown / CSV出力

## 前提

- [ ] Phase 0のGo条件を満たしている。
- [ ] Tauri + React + Viteの土台がある。
- [ ] SQLite migration方針が決まっている。
- [ ] Codex App Server stdio接続が動いている。
- [ ] 資料読み取りの最低品質が確認済み。
- [ ] Markdown / CSV出力を必須、PDFはレポート型1テンプレートに絞る方針が確定している。
