# ChatGPT device-code flow 説明文

作成日: 2026-05-16

## ユーザー向け手順

1. アプリのログイン画面で`ログイン手順確認`を押す。
2. 画面に表示されたURLをブラウザで開く。
3. 画面に表示されたコードをブラウザ側に入力する。
4. ChatGPTアカウントでログインし、認証を許可する。
5. アプリに戻り、接続状態が有効になることを確認する。

## 画面に出す情報

- verification URL
- user code
- 待機中の状態
- キャンセル結果
- 認証完了または失敗の結果

## Windows向け補足

- URLは既定ブラウザで開く。
- コードはコピーしやすい形で表示する。
- 会社PCでブラウザ制限がある場合は、別ブラウザまたは管理者設定が必要になる可能性がある。
- 30秒以上待っても戻らない場合は、アプリ側でキャンセルして再試行できるようにする。

## Phase 0確認範囲

- `account/login/start`の`chatgptDeviceCode`でURL/code発行を確認済み。
- 待機中にもTauri event経由でURL/codeをUIへ表示できる。
- `account/login/cancel`でキャンセル結果を表示できる。
- 実ログイン完了通知は、今回の自動検証では未確認。
